Risk Level Guide

High Risk

Unauthorized fund movement, exposed seed/private keys, confirmed backdoor behavior, or repeated victim reports.

Before You Buy a Solana Bot or Bundler Tool

Use this checklist before paying any Telegram seller, importing wallets, or granting server access.

01Search the developer name, handles, GitHub, wallet address, and tool name before payment.
02Never enter seed phrases or high-value private keys into unknown tools.
03Use a fresh VPS and fresh test wallets for first-run validation.
04Audit outbound network calls, wallet storage, and hidden API endpoints.
05Rotate all API, RPC, dashboard, and server credentials after developer access.
06Check authorized_keys, unknown users, open ports, processes, cron jobs, and modified files.

Recommended Safety Actions

If you used a suspicious Web3 tool or gave developer access, act before adding real funds.

✓Move funds to fresh wallets you created independently.
✓Treat imported or tool-generated wallets as potentially compromised until verified.
✓Remove unknown SSH keys and disable unnecessary developer access.
✓Back up evidence before links, repos, messages, or accounts disappear.
✓Publish neutral, evidence-based reports using alleged / reported / under review wording.
✓Report malicious repos, phishing pages, and scam accounts through official platform channels.
